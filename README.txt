РЕДИБРОКЕР — квиз-лендинг

Как открыть:
1. Распакуйте архив.
2. Откройте index.html в браузере.

Для публикации:
- GitHub Pages: загрузите все три файла в корень репозитория.
- Netlify/Vercel: перетащите папку проекта.
- Обычный хостинг: загрузите файлы в public_html.

Важно:
Форма сейчас демонстрационная. Для реального получения заявок нужно подключить Telegram-бота, CRM, email или серверный обработчик.
